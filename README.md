# rustex
Create pdf and beamer presentation like in latex.
Build your own setup to quickly create your new presentations.

## Build
```
./compile $ARG
xdg-open main.pdf
```

`$ARG`: default(debug), release

## Usage
Dimension in `.env`.

### Slides
Slides are located in `rsc/slide`.

Title page example provided.
